Скрипт используется для слежения за состоянием выбранных(scriptMonitor.csv) бумаг: 
имя бумаги
% изменения (выделяется цветом)
текущая цена
минимальная сегодня цена
максимальная сегодня цена
процент изменения за 5 дней (выделяется цветом)

позиция по бумаге (выделяется цветом)
цена приобретения
стоимость позиции в рублях
прибыль/убытки в рублях
прибыль/убытки в процентах

можно купить
можно продать

суммарный спрос
суммарное предложение
(большее выделяется цветом)

Кодировку UTF-8 можно поменять на Windows-1251 с помщью Notepad++.
Положение и размер окна задаются в строке №179.
Идущие часы в заголовке окна говорят о работающем скрипте.
